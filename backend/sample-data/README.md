# Sample API Data

1. Create a Vehicle.
2. Copy the returned Vehicle _id.
3. Replace `PUT_VEHICLE_ID_HERE` in:
   - trip.json
   - maintenance.json
   - fuelLog.json
   - expense.json

4. Create a Driver.
5. Copy the returned Driver _id.
6. Replace `PUT_DRIVER_ID_HERE` in trip.json.

Now all APIs are ready for testing in Postman.
